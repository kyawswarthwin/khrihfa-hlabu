KHRIH LE KHRIHFABU CAAH

Doh is F

1. Khrih le Khrihfabu ca ah, Lungkhat te in kan dir; A thangthat tu kan hmur, Sifak bawmtu kan kut; Mitlau kawltu kan ke, Khrih ca ah an si.

CHO: Khrih le Khrihfabu ca ah, Lungkhat te in kan dir; Bawi Khrih le Khrihfabu ca ah, Kan nunnak kan pek.

2. Khrih le Khrihfabu ca ah, Thlacam in rian kan tuan; Ngamh le lungtho ngai in, A rianthiang kan tuan lai; Kan biakamnak fek seh, Khrih ca ah ti in.

CHO: Khrih le Khrihfabu ca ah, Lungkhat te in kan dir; Bawi Khrih le Khrihfabu ca ah, Kan nunnak kan pek.

3. Khrih le Khrihfa-bu ca ah, Ho-sa-na hla kan sa; A kan luatter hnu cun, Teinak bia a kan kamh; "Khrih le Khrihfabu ca," Kan lungtum a si.

CHO: Khrih le Khrihfabu ca ah, Lungkhat te in kan dir; Bawi Khrih le Khrihfabu ca ah, Kan nunnak kan pek.
