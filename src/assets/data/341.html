ZUMHNAK HI TEINAK A SI

Doh is F

1. Nannih Khrihfa ralkap vialte, Tho u, hmaiah kal u, Ni ceunak kha zan muihnak nih, A khuh hlanah kal u, Kannih kan ral tihnungmi kha, Kan celh chung do u si. Zumhnak hi vawlei a teitu, Asi kha kan hngalh ko.

CHO: Zumh hi teinak a si, Zumh hi teinak a si (Zumh hi teinak a si) Vawlei teitu a si ko, Teinak lianngan bik cu.

2. Dawtnak kan hmelchunhnak a si, Pathian Bia kan vainam, Mithiang mi nih an zulhmi lam, Kannih zong kan zul ve, Zumhnak in khua zakip ah khan, Phuchim bangin an kal. Thihnak an teinak zumhnak kha, Kan phaw a si ve ko.

CHO: Zumh hi teinak a si, Zumh hi teinak a si (Zumh hi teinak a si) Vawlei teitu a si ko, Teinak lianngan bik cu.

3. Sualnak ral kha a teitu cu, Puanrang kha an pek lai, Vancung mithiang mi hna hmaiah, A min kha an chim lai. Khi tikah ceunak tlang cung khan, Hmailei ah kan kal lai, Muihnak ralkap kan tei hna lai, Kan Bawi Jesuh min in.

CHO: Zumh hi teinak a si, Zumh hi teinak a si (Zumh hi teinak a si) Vawlei teitu a si ko, Teinak lianngan bik cu.
