THLAITHAR LAWMHNAK THLUACHUAH

Doh is Db.

1. Fing le tlang sang, nelrawn, zo le chim hmunkip ah, Kan nunnak caah thlai thlak in kan tuan, Thei in a par tertu, Zeizong sertu Bawipa sinah lawmhnak langter philh hin hlah u.

CHO: Lo le dum chung chuakmi kan thlaithar vialte cu Petu Bawipa hmaiah ah chuahpi cio u, Thlaithar lawmhnak le thlaithar thluachuahnak Bawipa nih zungzal kan pe ko seh, Thlaithar lawmhnak le thluachuah chuahnak Bawipa nih, zungzal kan pe ko seh, A thluachuahnak kan sinah um ko seh.

2. Tuanfak le retheimi kan hna a ngamtertu, Kan tawkzawn in thlua chuah a kan petu, Zeizong ngeitu Bawipa, Khua-ruahhar dawtnak nih nifa kan herhmi cu a kan pek tawn.

CHO: Lo le dum chung chuakmi kan thlaithar vialte cu Petu Bawipa hmaiah ah chuahpi cio u, Thlaithar lawmhnak le thlaithar thluachuahnak Bawipa nih zungzal kan pe ko seh, Thlaithar lawmhnak le thluachuah chuahnak Bawipa nih, zungzal kan pe ko seh, A thluachuahnak kan sinah um ko seh.

3. Nitlang le van-rialruah kan herhmi khuati tha, A kan serpiaktu Bawipa thangthat u, Kum khat chung kan thlawh tuan than laimi lo le dum cinthlak vialte thei in parter ko seh.

CHO: Lo le dum chung chuakmi kan thlaithar vialte cu Petu Bawipa hmaiah ah chuahpi cio u, Thlaithar lawmhnak le thlaithar thluachuahnak Bawipa nih zungzal kan pe ko seh, Thlaithar lawmhnak le thluachuah chuahnak Bawipa nih, zungzal kan pe ko seh, A thluachuahnak kan sinah um ko seh.
