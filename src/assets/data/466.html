MO PUAI HLA

Doh is A

1. Maw kan Bawipa hi hmun thiang ah thitum biakam feh ter awkah kan ra, Kan cohlang law, samhri sih in, Nun chung nguh in biakam fek kan tuah lai.

CHO: Na dawtnak hri in run kan tem law, hohmanh nih then khawh lomi kan siter, Na van thluachuah run kan pe ko. Aa dawhmi inn-chungkhar si ve dingin.

2. Hi vawlei ah kan ton laimi zeizong kip ah kan i dawtnak hmun seh, Zawnruah dawtnak lungthin karh in, Dawt tlak lomi daw khotu kan siter.

CHO: Na dawtnak hri in run kan tem law, hohmanh nih then khawh lomi kan siter, Na van thluachuah run kan pe ko. Aa dawhmi inn-chungkhar si ve dingin.

3. Maw kan Bawipa na hmai ka ah thit-umnak cu hmaizahnak in kan tuah, Kan cohlang in, kan thianter ko, Vawlei thawkka i na timh cia bang khan.

CHO: Na dawtnak hri in run kan tem law, hohmanh nih then khawh lomi kan siter, Na van thluachuah run kan pe ko. Aa dawhmi inn-chungkhar si ve dingin.
