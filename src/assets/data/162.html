KAN HNGAH LIO JESUH A KAN HMU LAI MAW?

Doh is Bb

1. Bawi Jesuh a rung tum than te tikah, Chun, zan zei caan ti loin, Lung dinnak in kannih nih kan hngak maw? Meifar vialte timh cia in.

CHO: Maw kannih, vialte kan i tim cang maw? Thlarau tha umnak inn ca, Kan hngah lio Jesuh a kan hmu lai maw? Hngak ko, Jesuh tumnak kha hngak ko.

2. Pakhat hnu pakhat a kan auh tikah, Zinglei khua a dei lioah, A kan pekmi chaw kan pek than tikah, A tha, a kan ti lai maw?

CHO: Maw kannih, vialte kan i tim cang maw? Thlarau tha umnak inn ca, Kan hngah lio Jesuh a kan hmu lai maw? Hngak ko, Jesuh tumnak kha hngak ko.

3. A kan chimh takmi kha kan tuah cang maw? Kan tha a um chung vialte, Mawhchiatnak in kannih kan luat ahcun, Hnangamnak kha kan hmuh lai.

CHO: Maw kannih, vialte kan i tim cang maw? Thlarau tha umnak inn ca, Kan hngah lio Jesuh a kan hmu lai maw? Hngak ko, Jesuh tumnak kha hngak ko.

4. A hngak vialte thluachuahnak um ko seh, Liannak chung an um te lai. Chun, zan, le zantim caan zongah siseh, Kan hngahnak a hmu lai maw?.

CHO: Maw kannih, vialte kan i tim cang maw? Thlarau tha umnak inn ca, Kan hngah lio Jesuh a kan hmu lai maw? Hngak ko, Jesuh tumnak kha hngak ko.
