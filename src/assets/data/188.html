NITLANNAK AH KAN KAL

Doh is Bb

1. Nitlannak ah kan kal, dawh in a cer chin, Zing van sen le zan dapti lakah kan kal, Fek te-in hmailei nifatin in kan kal, Lam thluangthluan Jesuh zul tuah.

CHO: Zul tuah, Jesuh kha kan zulh lai, Zul tuah, Nifate zul tuah, Hmai ah a kan hruainak kip ah, Lam thluangthluan Jesuh cu kan zul lai.

2. A ceu bik nitlang le a muibik zantim, Hring dildel tlim-no le thal vulvau par caan, Zei caan paoh ah Jesuh fialmi cu zul tuah, Lam thluangthluan Jesuh zul tuah.

CHO: Zul tuah, Jesuh kha kan zulh lai, Zul tuah, Nifate zul tuah, Hmai ah a kan hruainak kip ah, Lam thluangthluan Jesuh cu kan zul lai.

3. Aa dawh bik nitlang le a muimi thladem, Thlichia to-tho hrang in hnabei dongh tikah, nitlak khuamuih zongah A-mah i bochan, Lam thluangthluan Jesuh zul tuah.

CHO: Zul tuah, Jesuh kha kan zulh lai, Zul tuah, Nifate zul tuah, Hmai ah a kan hruainak kip ah, Lam thluangthluan Jesuh cu kan zul lai.
