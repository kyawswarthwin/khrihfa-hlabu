PATHIAN NIH AN ZOH LAI

Doh is Bb

1. Sifah innak ca thinphang hlah, Pathian nih an zoh lai, Amah dawtnak thla tang um tuah, Pathian nih an zoh lai.

CHO: Pathian nih an zoh lai, Nifa kalnak khuallam ah khan, Amah nih an zohkhenh, Pathian nih an zoh lai (an zoh lai)

2. A fakpi rian na tuan tikah, Pathian nih an zoh lai, Na kalnak lam phannak um lo, Pathian nih an zoh lai.

CHO: Pathian nih an zoh lai, Nifa kalnak khuallam ah khan, Amah nih an zohkhenh, Pathian nih an zoh lai (an zoh lai)

3. Na duh vialte kha an pek lai, Pathian nih an zoh lai, Pek lo in zei hmanh um lai lo, Pathian nih an zoh lai.

CHO: Pathian nih an zoh lai, Nifa kalnak khuallam ah khan, Amah nih an zohkhenh, Pathian nih an zoh lai (an zoh lai)

4. Tukforhnak kha na hmuh tikah, Pathian nih an zoh lai, Amah tangchung ah va um tuah, Pathian nih an zoh lai.

CHO: Pathian nih an zoh lai, Nifa kalnak khuallam ah khan, Amah nih an zohkhenh, Pathian nih an zoh lai (an zoh lai)
