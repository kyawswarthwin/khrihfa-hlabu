RAM CEU DAWH

Doh is G

1. Ni khatkhat cu Pa-ra-dise ram tlangdawh lei cu kan hei cuan tawn, Ni khatkhat cu kan lung chungah, Pangpar rimhmui bangin aa zel, Ni khatkhat cu ka lung chungah liam cia caan an hung chuak ve tawn, Ni khatkhat cu kan lung chungah liamcia caan an hung chuak ve tawn, Dawtmi hna he zatial tlaih in, Ram ceu dawh ah kan leng ti lai, Ram ceu dawh ah kan leng ti lai, Zatial tlaih in kan leng ti lai, Ram ceu dawh ah hmunkhat te ah, Ram ceu dawh ah, ram ceu dawh ah.

2. Ni khatkhat cu kan hnathlam ah sual teinak hla a hung cuang tawn, Ni khatkhat cu kan lung chungah, Philh khawh lomi a hung chuak tawn, Ni khatkhat cu hi thil vialte, dimhnak hmun ah kan hmuh te lai, Ni khatkhat cu hi thil vialte, dinhnak hmun ah kan hmuh te lai, dinhnak hmun ah kan hmuh te lai, Maw hawi le hna hngak ko rih u, Khi caan tha a phanh lai kan hngalh, Khi caan tha a phanh lai kan hngalh, Maw hawidawt hna hngak ko rih u, Khi caantha a phanh lai kan hngalh, Ram ceu dawh ah, ram ceu dawh ah.
