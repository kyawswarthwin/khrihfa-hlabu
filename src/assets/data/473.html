ZUMH IN KAN KAL

Doh is Bb

1. Zumh in kan kal, Zeitluk nuam dah! Par dawh an enhnak ah kan kal, Kan lam thluan ah, Rimhmui a zel, Nun dongh lonak lei cu kan panh.

CHO: Zum in kan kal, Tuukhal tha he, A aunak aw nem cu kan theih, Kan kut tein a kan hruai i, Kan pawng a um zungzal kan hngalh.

2. Zumh in kan kal, A bia bangin, Kan kalnak lam, a kan hngalhter, Khi lioah kan lam mui hmanhseh, Amah pawng ah cun a kan hruai.

CHO: Zum in kan kal, Tuukhal tha he, A aunak aw nem cu kan theih, Kan kut tein a kan hruai i, Kan pawng a um zungzal kan hngalh.

3. Zumh in kan kal, Thlua-chuahnak he, Amah sin ah cun kan i din, Tuukhal tha kan i bochan i, A dawtnak cu kan i hrawm cio.

CHO: Zum in kan kal, Tuukhal tha he, A aunak aw nem cu kan theih, Kan kut tein a kan hruai i, Kan pawng a um zungzal kan hngalh.

4. Zumh in kan kal, nunnak dongh tiang, Hawidawt hna he kan leng ti lai, Ceunak hmun ah, him in kan um, Zumhnak cu mithmuh ah a tling.

CHO: Zum in kan kal, Tuukhal tha he, A aunak aw nem cu kan theih, Kan kut tein a kan hruai i, Kan pawng a um zungzal kan hngalh.
