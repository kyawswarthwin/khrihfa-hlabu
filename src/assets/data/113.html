A KAN KHAMH CANG

Doh is Eb

1. A kan khamh cang, khuaruahhar in, A kan Khamhbawi nih a thi cang; Rulhreu temnak a phoih dih cang. Bawi Jesuh nih a kan khamh cang.

CHO: A kan khamh cang, a kan khamh cang, A thlet thi in a kan tawl cang, Vailamtah cung thihnak a in; A kan Khamhbawi kei ca a thi.

2. A kan khamh cang, rulhreu temnak, A phoih dih cang keimah cungah; A kan Khamhbawi thangthat u si. A vailam kha pu cio u si.

CHO: A kan khamh cang, a kan khamh cang, A thlet thi in a kan tawl cang, Vailamtah cung thihnak a in; A kan Khamhbawi kei ca a thi.

3. Khamh a duh vial a khamh dih cang, Khuaruahharnak hlasa u si; Tingtang tum in thangthat u si. Khamhbawi min kha thangthat u si.

CHO: A kan khamh cang, a kan khamh cang, A thlet thi in a kan tawl cang, Vailamtah cung thihnak a in; A kan Khamhbawi kei ca a thi.
