JESUH KAN HAWI THA BIK A SI

Doh is F

1. Kannih kan sual sifah ingtu, Kan hawi Jesuh a si lai, Pathian sinah kan duh vialte Hal ah cun a tha dah ngai, Hna-ngamnak tampi um loin, Fahnak ngaihchiatnak kan huah, Kan duh vialte Pathian sinah, Hal loin kan um caah.

2. Fahnak le tukforh kan huah maw, Hna-hnawhnak in kan um dah? Kan lung belte rawk hin hlah seh, Pathian sinah thlacam u, Sifah huahnak kha an chanhtu, Hawi khuazei ah na hmu maw, Kan ngaih-chiatnak kha Jesuh hngal, Pathian sinah thlacam u.

3. Zaang ngei lomi kan si cio maw, Kan thil kha a rit tuk dah? Asungmi kan Khamhtu Jesuh, A ngaih lai, kan hal tikah. Kan hawi nih an kan hlawt ahcun, Pathian Bawi sinah chim u, Akut in an lak an pom lai, Hna-ngamnak kan hmuh lai ta.
