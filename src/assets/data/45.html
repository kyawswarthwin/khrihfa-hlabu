THLICHIA A HRAN AH KAN DORNAK

Doh is F

1. Kan himnak lung Jesuh a si, Thlichia a hran ah kan dornak, Harnak zeikhom in kan khamh rih, Thlichia a hran ah kan dornak,

CHO: Maw, retheimi caah Jesuh kan lungpi, Retheimi ca lungpi a si, Maw, retheimi caah Jesuh kan lungpi, Thlichia a hran ah kan dornak.

2. Chun ah naidem, zan ah dornak, Thlichia a hran ah kan dornak, Tihnung ral nih kan doh hmanhseh, Thlichia a hran ah kan dornak,

CHO: Maw, retheimi caah Jesuh kan lungpi, Retheimi ca lungpi a si, Maw, retheimi caah Jesuh kan lungpi, Thlichia a hran ah kan dornak.

3. Thlichia nih a kan dum tikah, Kan dornak kha Jesuh a si, Himnak hmun in kan zam lai lo, Thlichia a hran ah kan dornak,

CHO: Maw, retheimi caah Jesuh kan lungpi, Retheimi ca lungpi a si, Maw, retheimi caah Jesuh kan lungpi, Thlichia a hran ah kan dornak.

4. Maw, kan lungpi, himnak fek bik, thlichia a hran ah kan dornak, Kannih naih ah zungzal um tuah, Thlichia a hran ah kan dornak.

CHO: Maw, retheimi caah Jesuh kan lungpi, Retheimi ca lungpi a si, Maw, retheimi caah Jesuh kan lungpi, Thlichia a hran ah kan dornak.
