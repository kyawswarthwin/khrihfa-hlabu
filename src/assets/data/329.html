THLUACHUAHNAK RUAHPI A SUR LAI

Doh is Bb

1. Thlua-chuahnak ruahpi a sur lai, Dawtnak biakam nak bang in, Cawrh than caan kha a tlung te lai, Jesuh sin in a tlung lai.

CHO: Thluachuah ruahsurnak Thluachuah ruahsurnak kan duh, Velngeih ruah a sur ciah-mah cang, Ruah sur tamnak kan duh chin.

2. Thlua-chuahnak ruahpi a sur lai, A uaimi aa tung than lai, Ruahpi a sur thawng kan theih lai, Tlangcung le tlangkar ah khan.

CHO: Thluachuah ruahsurnak Thluachuah ruahsurnak kan duh, Velngeih ruah a sur ciah-mah cang, Ruah sur tamnak kan duh chin.

3. Thlua-chuahnak ruahpi a sur lai, Pathian atu kan pe tuah, Nangmah biakamnak a si ca, Tun thannak kha ra kan pe.

CHO: Thluachuah ruahsurnak Thluachuah ruahsurnak kan duh, Velngeih ruah a sur ciah-mah cang, Ruah sur tamnak kan duh chin.

4. Thlua-chuahnak ruahpi a sur lai, Nihin zong sur seh kan duh, Kan sualnak kha kan phuan lioah, Jesuh sinah thla kan cam.

CHO: Thluachuah ruahsurnak Thluachuah ruahsurnak kan duh, Velngeih ruah a sur ciah-mah cang, Ruah sur tamnak kan duh chin.
