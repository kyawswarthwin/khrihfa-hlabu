A MAN A SUNG

Doh is A

1. Kan lungthin Bawipa Biakinn thiang a si tikah, A chungah Bawipa a um, Thlacamnak thiang he a min in kan i pum i, Zeitluk in dah kan i lawmh.

CHO: A zummi hna caah cun a sunglawi bik, A cohlangmi hna caah a sunglawi bik; A min in kan i pum i, thluachuahnak kan hal lioah, Kan thlacamnak a cohlan.

2. A sunparnak chungin ceunak phun tling a tum, Kan ruahchannak zor hmanh seh, A dawtnak a ngan ruangah teinak kan hmuh lai, Amah he kan um ca ah.

CHO: A zummi hna caah cun a sunglawi bik, A cohlangmi hna caah a sunglawi bik; A min in kan i pum i, thluachuahnak kan hal lioah, Kan thlacamnak a cohlan.

3. A kan hruainak kan zulh i kan hna a ngamter, Kan chungah lawmhnak a khat, A hmunmi a kut in a kan huhphenh zungzal, ti cu fiang tein kan zumh.

CHO: A zummi hna caah cun a sunglawi bik, A cohlangmi hna caah a sunglawi bik; A min in kan i pum i, thluachuahnak kan hal lioah, Kan thlacamnak a cohlan.

4. Biatak hngalhnak in a vel chungah thang u si, Hnangamnak tling ngei u si. Siangpahrang inn ah zatial tlaih in kan leng lai, Lawmhnak a dong ti lai lo.

CHO: A zummi hna caah cun a sunglawi bik, A cohlangmi hna caah a sunglawi bik; A min in kan i pum i, thluachuahnak kan hal lioah, Kan thlacamnak a cohlan.
