KAN IN THANGTHAT KAN PATHIAN

Doh is F

1. Kan in thangthat, Pathian Kan lungthin le kan aw in, Khuaruahhar thil tuahtu, Vawlei mi nih an thangthat; Leicung kan chuahka in thluachuah a kan petu, Rel cawk lo dawtnak he Nihin ni tiang kan Pa.

2. Zeizong tlingmi Pathian, Kan sinah um zungzal tuah, Donglo lawmhnak thinlung, Daihnak he kan hnem ko tuah, Kan huppheng zungzal law kan hruai lam kan tlau ah, Lei le vancung zongah, Temhnak in kan luat-ter.

3. Thangthat lawmhnak vialte, Kan Pa Pathian pe ko u, Amah sin a ummi Fapa le Thiangthlarau he, Zungzal hmunmi Pathian lei le van nih biakmi, Zungzal in zungzal tiang A hmun zungzal ko lai.
