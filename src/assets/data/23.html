PATHIAN NIH AN ZOH LAI

Doh is Db

1. Pathian nih an zoh lai, na thinphang hlah, Ceu le mui ah an hupphengtu a si, A mi hna cu zungzal in a ven hna, Umhar lileng in an chia ta lai lo.

CHO: Pathian nih an zoh lai, A dongh tiangin, Hawikom tha bik, Tlantu kan Pa a si, Na auh tikah Jesuh nih an leh lai. Va zum zungzal, Amah nih an zoh lai.

2. Pathian nih an zoh lai, chun nitlak in, Na kehlan cu lam dingah a hruai lai, Tuukhal bangin an hupphengtu a si, Cerhti kamah him in a hruai tawn hna.

CHO: Pathian nih an zoh lai, A dongh tiangin, Hawikom tha bik, Tlantu kan Pa a si, Na auh tikah Jesuh nih an leh lai. Va zum zungzal, Amah nih an zoh lai.

3. Pathian nih an zoh lai, chan chung nguh in, Hmuh bal lomi a thluachuah an pek lai, Caan dong hmanh seh zungzal in an zohlai, Hnangam tein a sinah an auhlai.
