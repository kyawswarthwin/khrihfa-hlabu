NICHUAHLEI SIANGPAHRANG PATHUM

Doh is G

1. Nichuahlei Siangpahrang kannih, Pathum nih khual kan tlawn pah in, Lo le cerh le fing le tlang ah, Arfi kan zulh lengmang.

CHO: Maw, khuaruahharmi zan arfi, Aa dawh ngaingaimi arfi, Nitlaklei ah kan hruai lengmang, Na ceunak kan phak hlan tiang.

2. Bethlehem chuakmi Siangpahrang, Chinh thanawk sui ka rak i put, Dong hin loin kan siangpahrang, Kannih kan umpi tuah.

CHO: Maw, khuaruahharmi zan arfi, Aa dawh ngaingaimi arfi, Nitlaklei ah kan hruai lengmang, Na ceunak kan phak hlan tiang.

3. Pek dingah frenkensens ka ngei, Pathian biaknak caah zihmui, Mi vialte nih an thangthat i, Lianngan Pathian an biak.

CHO: Maw, khuaruahharmi zan arfi, Aa dawh ngaingaimi arfi, Nitlaklei ah kan hruai lengmang, Na ceunak kan phak hlan tiang.

4. Cinthling kei ka laksawng a si, Hi zihmui rim a khami hi; Ngaihchiat khuaruahhar le thi chuah, Hmelchunhnak cu a si.

CHO: Maw, khuaruahharmi zan arfi, Aa dawh ngaingaimi arfi, Nitlaklei ah kan hruai lengmang, Na ceunak kan phak hlan tiang.

5. Zohhmanh, thangthatnak in a tho, Pathian kannih kan Khamhtu cu, Hallelujah, Hallelujah, Van le lei ah a thang.

CHO: Maw, khuaruahharmi zan arfi, Aa dawh ngaingaimi arfi, Nitlaklei ah kan hruai lengmang, Na ceunak kan phak hlan tiang.
