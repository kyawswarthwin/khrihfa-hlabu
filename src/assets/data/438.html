NI BANG A CEUMI KHUA

Doh is Bb

1. Ni bang a ceumi khua a si, Vancung khuatha chung khi, Nitlak lonak ram ceunak cu, Khamhtu Jesuh a si.

CHO: A vailam kan put ahcun, (A vailam kan put ahcun), Sui luchin kan hmuh lai, (Sui luchin kha kan hmuh lai), Vancung khua kan phak tikah, (Vancung khua kan, phak tikah), A zungzal in kan ceu lai.

2. Vancung khua ah hnangam a um, Thihnak tiva ral ah, Tihnung vialte a lohnak ah, Khi khin kan ro a um.

CHO: A vailam kan put ahcun, (A vailam kan put ahcun), Sui luchin kan hmuh lai, (Sui luchin kha kan hmuh lai), Vancung khua kan phak tikah, (Vancung khua kan, phak tikah), A zungzal in kan ceu lai.

3. Pathian remhmi inn dawh bik cu, Mithiang umnak a si, Kan dawt ngaingai mi hna nih khan, Khikhin an kan don lai.

CHO: A vailam kan put ahcun, (A vailam kan put ahcun), Sui luchin kan hmuh lai, (Sui luchin kha kan hmuh lai), Vancung khua kan phak tikah, (Vancung khua kan, phak tikah), A zungzal in kan ceu lai.

4. Vawlei cung tih-phannak vialte, Zokzok in lonh ka duh, Vancung mithiang mi sinah khin, Thihnak a um lai lo.

CHO: A vailam kan put ahcun, (A vailam kan put ahcun), Sui luchin kan hmuh lai, (Sui luchin kha kan hmuh lai), Vancung khua kan phak tikah, (Vancung khua kan, phak tikah), A zungzal in kan ceu lai.
