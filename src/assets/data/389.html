SEINO KAN SI LIOAH

Doh is Eb

1. Seino kan si lioah, Kan thazang tam lioah, Nang kan nunnak kan Bawipa cu, Kan lungthin kan in pek, Kan laksawng cohlang tuah, Na duh bang kan siter, Kan Pa duhnak tlinter awkah, Kan nunchung kan bawm tuah.

2. Khoika kan um hmanh ah, Nangmah duhnak tuah le, Na hmaika i um bantukin, Lungthin kan ngeihter tuah, Thil ngan thim theng loin, Thil hme nautat loin, Kan rian paoh cu nangmah kut in, Kan hmuh, kan hngalhter tuah.

3. Biatak le lungthin thiang, Kan duh nakhnga kan bawm, Caan tawite hmanh ruahnak tha lo, Kan ngeih nakhnga lo le, Raltha le tih loin, Kan dir nakhnga kan bawm, A thiangmi thil a duhtu le, Thiang lo hua si kan bawm.

4. Bawi Jesuh Khrih Thlarau, Kan lungchung ceuter tuah, A thiang le dawtnak nunnak lam, Kan zulh khawh nakhnga in, Kan hmailei kum caah, Nangmah nih kan tim ko, Kan Pa ram chung zungzal nunnak, Kannih kan phak hlan tiang.
