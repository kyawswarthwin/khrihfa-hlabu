ZUN AH ZEITINDAH SI LAI?

Doh is C

1. Khuadei ah lu-nunnak ci kan vorh, Chun ah Pathian lu nunnak ci kan vorh, Zanlei ah lu-nunnak ci kan vorh, Zantim ah lu-nunnak ci kan vorh, Zun ah zeitindah lai, Zun ah zeitindah si lai?

CHO: Zan ah kan vorh siseh, Chun ah kan vorh, Zaang-hrim le, zaang cham ah kan vorh si seh, A zuntu vanmi an si ca-ah, Vancung thlaici kha vorh ko u.

2. Lam pawng lu-nun thlaici kan vorh ah, Lung lakah lu-nun thlaici kan vorh ah, Hling lak lu-nun thlaici kan vorh ah, Vawlei tha ah thlaici kan vorh ah. Zun ah zeitindah lai, Zun ah zeitindah si lai?

CHO: Zan ah kan vorh siseh, Chun ah kan vorh, Zaang-hrim le, zaang cham ah kan vorh si seh, A zuntu vanmi an si ca-ah, Vancung thlaici kha vorh ko u.

3. Lungfah in sualnak ci nan vorh ah, Lungrawh in sualnak thlaici nan vorh ah, Min chiat in sualnak ci nan vorh ah, ningzah in sualnak ci nan vorh ah, Zun ah zeitindah lai, Zun ah zeitindah si lai?

CHO: Zan ah kan vorh siseh, Chun ah kan vorh, Zaang-hrim le, zaang cham ah kan vorh si seh, A zuntu vanmi an si ca-ah, Vancung thlaici kha vorh ko u.
