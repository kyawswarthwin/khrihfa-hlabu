NGAIHMANH, PATHIAN DAWTNAK CU

Doh is Db

1. Ngai hmanh u, Pathian dawtnak cu, Khuaruahhar ngai a si, Pathian fa ah misual hmanh hi, A kan canter dah ngai.

CHO: Ngai hmanh! Khuaruahhar dawtnak, Pathian a kan dawtnak cu zeitluk in dah a lianhngan! Pathian fa a kan ti cang. A khamhmi kan si caah.

2. Kan ca a thi a thlet ca ah, Khamhbawi he kan i naih, Ka fa dawtmi hna tiah a au, A tang ah kan um lai.

CHO: Ngai hmanh! Khuaruahhar dawtnak, Pathian a kan dawtnak cu zeitluk in dah a lianhngan! Pathian fa a kan ti cang. A khamhmi kan si caah.

3. Kan sunpar nak zei bantuk dah, Atu a lang rih lo. Khamhbawi a langh than te tikah, Amah bang kan si lai.

CHO: Ngai hmanh! Khuaruahhar dawtnak, Pathian a kan dawtnak cu zeitluk in dah a lianhngan! Pathian fa a kan ti cang. A khamhmi kan si caah.

4. Khi bia a zummi vialte cu, Khamhbawi bang kan si lai, Thihnak teitu minthang Khamhbawi, Na hmai hmuh ka duh ko.

CHO: Ngai hmanh! Khuaruahhar dawtnak, Pathian a kan dawtnak cu zeitluk in dah a lianhngan! Pathian fa a kan ti cang. A khamhmi kan si caah.
