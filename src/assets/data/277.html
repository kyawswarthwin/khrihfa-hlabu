ZION KHUA AH KAN KAL LIO

Doh is G

1. Jesuh a dawmi ra, Nan lawmhnak hngalhter tuah, Nannih dihlak hlasa cio u, Nannih dihlak hlasa cio u, Bawi thutdan pawngah cun, Bawi thutdan pawngah cun.

CHO: Zion khua kan kal lai, Zoh duh awk, aa dawh khua Zion Cunglei ah a ummi Zion, Pathian khua tha ah kan kal lai.

2. Pathian hngal lomi nih, Hlasak al hmanh hna seh, Vancung Bawibik fa hna nih cun, Vancung Bawibik fa hna nih cun, Hla cu an sak theo lai, Hla cu an sak theo lai.

CHO: Zion khua kan kal lai, Zoh duh awk, aa dawh khua Zion Cunglei ah a ummi Zion, Pathian khua tha ah kan kal lai.

3. Zion tlang cu a tha, Lawmhnak a tam a um, Vancung khua kha kan phak hlanah, Vancung khua kha kan phak hlanah, Kan hmuh nuamhnak a tam, Kan hmuh nuamhnak a tam.

CHO: Zion khua kan kal lai, Zoh duh awk, aa dawh khua Zion Cunglei ah a ummi Zion, Pathian khua tha ah kan kal lai.

4. Khi ca hlasa u si, mitthli kha a hul lai, Im-ma-nu-el ram in kan kal i, Im-ma-nu-el ram in kan kal i, Khua thabik kan phan lai, khua thabik kan phan lai.

CHO: Zion khua kan kal lai, Zoh duh awk, aa dawh khua Zion Cunglei ah a ummi Zion, Pathian khua tha ah kan kal lai.
