PATHIAN KAN SIN A TAN AHCUN

Doh is G

1. Bawipa cu lawm u, A vel-ngeihnak ruangah, Sual hremtu cu a thek dih hna, Thisen in a kan tlanh cang tih awk um lo, Jesuh cu kan zeizong a si.

CHO: Pathian cu (Pathian cu kan sin) kan sin a tan (a tan cang ahcun, A) cang ahcun Aho nih dah doh in an kan tei khawh hnga, Ho, ho? Ho (dah a kan tei hnga,) Ho nih dah an kan tei khawh ti hnga.

2. Bawipa bochan in, Nan tha tho chin ko seh, Nifa zumh-tlakmi si dingin, That-lonak lakah dinnak cu kan tanh lai, Amah cu kan thawnnak a si.

CHO: Pathian cu (Pathian cu kan sin) kan sin a tan (a tan cang ahcun, A) cang ahcun Aho nih dah doh in an kan tei khawh hnga, Ho, ho? Ho (dah a kan tei hnga,) Ho nih dah an kan tei khawh ti hnga.

3. A bia cu zum tuah, A biakamnak a hmun, Khrih cu zeizong vialte a si, Vawlei dong hmanhseh a bia zungzal a hmun, Bawipa biafiang tialmi an si.

CHO: Pathian cu (Pathian cu kan sin) kan sin a tan (a tan cang ahcun, A) cang ahcun Aho nih dah doh in an kan tei khawh hnga, Ho, ho? Ho (dah a kan tei hnga,) Ho nih dah an kan tei khawh ti hnga.

4. Bawipa chungah um, Him in an huhphenh lai, Zungzal nunnak caan a phan cang, A derthawm bik hmanh a kut in a huhphenh hna, Hohmanh nih an chut kho lai lo!

CHO: Pathian cu (Pathian cu kan sin) kan sin a tan (a tan cang ahcun, A) cang ahcun Aho nih dah doh in an kan tei khawh hnga, Ho, ho? Ho (dah a kan tei hnga,) Ho nih dah an kan tei khawh ti hnga.
