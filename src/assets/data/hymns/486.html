RO THA TANG SEH

Doh is Ab

1. Hlan lio kan pupa hna bangin lungkhat tein kan tuan hna lai, Zohchunh awk tlak ro tha tang seh, leile lung hmunh chung dir ve seh, Kan Khrihfabu dirkamh u si, Zohchunh awktlak ro tha tang seh.

CHO: Kan Bawipa nih a kan cah tak pakhat le khat run i daw u, kan pupa hna nih an zulh bang, lungtak tein kan zulh ve lai Kan nunnak hruaitu si seh, Bawi Khrih chungah pumkhat kan si, kan bia bangin hmun zungzal seh, Kan pupa keneh kan zulh lai, Kan pupa bangin kan tuan lai, Kan ram chungah dawtnak karh seh.

2. Kan Bawipa rovuih bia bangin pakhat le khat kan i daw lai, Zohchunh awk tlak ro tha tang seh, tuuphaw aih menmi si loin tuu lungthin tak in kan nung lai, Zohchunh awktlak, ro tha tang seh.

CHO: Kan Bawipa nih a kan cah tak pakhat le khat run i daw u, kan pupa hna nih an zulh bang, lungtak tein kan zulh ve lai Kan nunnak hruaitu si seh, Bawi Khrih chungah pumkhat kan si, kan bia bangin hmun zungzal seh, Kan pupa keneh kan zulh lai, Kan pupa bangin kan tuan lai, Kan ram chungah dawtnak karh seh.

3. Kan Bawipa lungput kan ngei lai, Toidor in mizawn kan ruat lai, Zohchunh awktlak ro tha tang seh, zerzau le sentai um loin Tleng khat thu va bang kan zuang lai. Zohchunh awktlak, ro tha tang seh.
