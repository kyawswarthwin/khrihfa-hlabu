MINMEI A TIAN DIH TIKAH

Doh is D

1. A dawh ngaingai mi tlangcung in Minmei kha a tian tikah, Cerhti le tiva te cungah, Ni a tlang, lawmhnak a tam Chuncha kan hmuh tikah Pathian Biakamnak cu kan hngalh than, Kannih kan i tonnak caan ah, Minmei kha a tian dih lai.

CHO: Hi vawlei (Hi vaw lei) kan i ton bang, (kan i tong bang) Vancung ah (Vancung ah) kan i tong lai, (kan i tong lai) Zinglei khua a ceunak chungah, Tha chin lawmhnak kan hmuh lai. Kannih kan i hmuhnak caan ah, Minmei kha a tian dih lai.

2. Mi vialte nih vawlei cungah Palhnak kha an tuah lengmang, Khuamui chung i rian kan tuan ah, Dawtnak bia kha kan hlawt tawn Kan Khamhtu nih, "Ra u law, Nan tuan man kan pek lai," a ti, Zing leiah kan i tong than lai, Minmei a tian dih tikah.

CHO: Hi vawlei (Hi vaw lei) kan i ton bang, (kan i tong bang) Vancung ah (Vancung ah) kan i tong lai, (kan i tong lai) Zinglei khua a ceunak chungah, Tha chin lawmhnak kan hmuh lai. Kannih kan i hmuhnak caan ah, Minmei kha a tian dih lai.

3. Minmei cu kannih cung leiah, A kai a loh dih tikah, Kan dawt kan hawi mi vialte he, Kannih kan i tong lai maw? Khua-muihnak ram tiva ral ah Khua ceunak kha a tlung cang; Khua-ruahhar a ceu kan hmuh lai, Minmei a tian dih tik ah.
