THLAICI ZUNTU AUHNAK

Doh is Ab

1. Khuazakip i rawl vialte cu, An tuan caan kha a cu cang, Tlang cung i le, nelrawn i zong, An za tein an hmin cang.

CHO: Maw, Kan Bawipa, kan in nawl ko, facang zuntu run thlah hna. A caan kha a luan dih hlanah, Rian rang in run thlah tuah hna.

2. Zing ah siseh, chun ah siseh, Ni a tlak dengmang zongah, Bawipa, hmunkip le ram kip ah, Rawltuan tu kha va fial hna.

CHO: Maw, Kan Bawipa, kan in nawl ko, facang zuntu run thlah hna. A caan kha a luan dih hlanah, Rian rang in run thlah tuah hna.

3. Maw nang, Bawipa nih, A thlahmi, Pum tuah sui facang phal cu, Na nunnak ni a liam tikah, Lawmhnak he naa din cang lai.

CHO: Maw, Kan Bawipa, kan in nawl ko, facang zuntu run thlah hna. A caan kha a luan dih hlanah, Rian rang in run thlah tuah hna.
