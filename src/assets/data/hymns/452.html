KHAMHBAWI NIH A KAN HRUAI LAI

Doh is Ab

1. Sifahnak kha kaltak in, A zungzal vancung khua in Vansan harnak um hmanhseh, Khamhbawi nih a kan hruai lai.

CHO: Zungzal van (Zungzal van inn khi) inn khi kannih (van inn khi kan naih), Kan Pa ram (Kan Pa ram thluachuah) thlua-chuahnak hmun (ram thluachuahnak hmun), Zungzal mit (Zungzal mitthli tlak) thli tlak lonak (mitthlitlak lonak) Vancung khuatha kan phak lai.

2. A thiangmi Tuufa umnak, Zion khua thar kha a lang, Amah thutnak hmun ah khin, Kan Khamhtu kha kan hmuh lai.

CHO: Zungzal van (Zungzal van inn khi) inn khi kannih (van inn khi kan naih), Kan Pa ram (Kan Pa ram thluachuah) thlua-chuahnak hmun (ram thluachuahnak hmun), Zungzal mit (Zungzal mitthli tlak) thli tlak lonak (mitthlitlak lonak) Vancung khuatha kan phak lai.

3. Harnak kha kan lonh tikah, Vancung mi van khua chungah, Ton kan duh kan ngai Khamhtu, Lung-lawmhnak in kan ton lai.

CHO: Zungzal van (Zungzal van inn khi) inn khi kannih (van inn khi kan naih), Kan Pa ram (Kan Pa ram thluachuah) thlua-chuahnak hmun (ram thluachuahnak hmun), Zungzal mit (Zungzal mitthli tlak) thli tlak lonak (mitthlitlak lonak) Vancung khuatha kan phak lai.

4. Kan ngai ti lo vawlei nun, Khamhbawi kha kan hmuh tikah, A sermi vancung khua thar, A sualmi kannih caah.

CHO: Zungzal van (Zungzal van inn khi) inn khi kannih (van inn khi kan naih), Kan Pa ram (Kan Pa ram thluachuah) thlua-chuahnak hmun (ram thluachuahnak hmun), Zungzal mit (Zungzal mitthli tlak) thli tlak lonak (mitthlitlak lonak) Vancung khuatha kan phak lai.
