JESUH, KA LUNGTHIN IN KAN DUH

Doh is C

1. Jesuh ka lungthin in kan duh, Na zaangfahnak kha kaa hmuh lai, Nangmah duh bangin ka ser ko, Nangmah lawngte kan dawt ko lai.

CHO: Ruahnak lung tha kha run ka pe, A thiang khua kha ka hmuhter tuah, Ka lung chung kha run ceuter law, Na umnak kei ca van a si.

2. Bawipa nangmah nih na kan dawt, Misual mi kha nang ca kan si, Thanuam in nangmah lei kan hoih, Na lawmhnak hmai kha kan hmuh lai.

CHO: Ruahnak lung tha kha run ka pe, A thiang khua kha ka hmuhter tuah, Ka lung chung kha run ceuter law, Na umnak kei ca van a si.

3. Lawmhnak phundang tam hmanhsehlaw, Pangpar tha kha hmu hmanhning law, Ka thlarau hlautu ah zeihmanh, Na zaangfahnak bantuk um lo.

CHO: Ruahnak lung tha kha run ka pe, A thiang khua kha ka hmuhter tuah, Ka lung chung kha run ceuter law, Na umnak kei ca van a si.

4. Nangmah loin ceunak um lo, Ka lungchung kha a ceutertu, Vawlei harnak ceuter ko tuah, Lawmhnak vialte nang ta an si.

CHO: Ruahnak lung tha kha run ka pe, A thiang khua kha ka hmuhter tuah, Ka lung chung kha run ceuter law, Na umnak kei ca van a si.
