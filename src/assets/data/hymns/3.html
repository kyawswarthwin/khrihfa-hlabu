SIANGPAHRANG BIA TUAH

Doh is A

1. Vancung Siangpahrang hmaizahnak pe u, A sunparnak cu hla in thangthat u; Amah cu kan phaw le kan himnak asi, Zungzal hmunmi sunparnak Pathian a si.

2. A thawnnak chim u, A velngeihnak zong, A puan ceunak nih zeizong a khuh dih. Vanrang le khuadawm cit in a ra lai i, Vawleicung mi vialte bia a ceih hna lai.

3. Na kan zohkhenhnak holh in chim khawh lo, Thli chungah a khat, ceunak chung zongah Tlangcung in a rung i tiva nelrawn tiang, Vanruah le dapti in dai tein an sur.

4. Ngakchia bantukin vawleicung mi nih, Nangmah kan in zumh, na kan hleng bal lo, Na velngeihnak cu zungzal in a hmun ko, A kan Sertu, Khamtu le kan Hawi na si.
