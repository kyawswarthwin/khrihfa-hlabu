A ZUNGZAL CA VOI KHAT LAWNG

Doh is Eb

1. Hmasa phungbia in luatnak cu a tha, Jesuh a thi thlet in khamhnak kan hmuh, Adam chiatserh sualnak ka chung in, Jesuh a kan khamh lai nihin.

CHO: A kan khamh cang, Maw misual zum tuah, A kan khamh cang, Unau zum tuah u, Hreng-khenh hngat in thilrit a thumh lai, Jesuh nih a kan khamh cang lai.

2. Sualnak in kannih nih kan luat cang lai, Jesuh khamhnak fiang tein a lang cang, "Ka sinah ra," ti in a kan auh, Ra tuah, atu a kan khamh lai.

CHO: A kan khamh cang, Maw misual zum tuah, A kan khamh cang, Unau zum tuah u, Hreng-khenh hngat in thilrit a thumh lai, Jesuh nih a kan khamh cang lai.

3. Pathian fale kan si ca lawmhnak in, Tukforhnak kha kan i sum kho lai ta, Thihnak kan in, lu nun kan hmuh cang, Lu nun khamhnak thluachuah a si.

CHO: A kan khamh cang, Maw misual zum tuah, A kan khamh cang, Unau zum tuah u, Hreng-khenh hngat in thilrit a thumh lai, Jesuh nih a kan khamh cang lai.
