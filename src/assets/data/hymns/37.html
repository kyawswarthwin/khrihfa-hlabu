LUNGLAWMHNAK IN KAN IN THANGTHAT

Doh is G

1. Lunglawmhnak in kan in thangthat, Sunpar Pathian dawtnak Bawi, Pangpar ni hoih in a par bang, Kan lung panh in umter ko, Sual le ngaihchiat minmei tianter, Zumh khawh lonak muihnak zong, a hmunmi lung lawmhnak petu, Ni ceunak run kan pe tuah.

2. Na sermi nih an in thangthat, Lei le van na cerh dih hna, Arfi le vanmi hla an sa, Thangthat dai lo Bawi na si, Lo le tupi, nel le fingtlang, Ramlak pangpar rili luang, va hramthiam le cerhti luang nih, Nangmah lawmh awk an kan sawm.

3. Leicung mi lawmhnak hla sa u, Deirel arfi an sakmi, Kan Pa dawtnak nih a kan pen, Unau dawtnih a kan tem, Hlasak in hmaiah kan fong chin, Harnak ah teitu kan si, ni lei ah hla nih a kan hruai, Teinak nunnak hla a si.
