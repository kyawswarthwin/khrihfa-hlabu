THIANG THLARAU LAM HMUHSAKTU

Doh is G

1. Thiang Thlarau lamhmuh-saktu, Khrihfa pawngah um zungzal, Kan kut in nifa hruai tuah, Vancung khua kan kai lioah, Thawngtha bik kan theih tikah, Retheimi an tha a nuam, "Na lam kha kan hmuhsak lai, Zul tuah," bia nem in a ti.

2. Keimah he um ti ka hawi, Keimah bawmtu ka Khamhbawi, Khuamui ah kanchia ta hlah, Thinphannak kan huah tikah, Thlitu hrang ruahpi sur i, Kan ruahchannak lung a dong, "Na lam kha kan hmuhsak lai, Zul tuah," bia nem in a ti.

3. Riantuannak a dih tikah, Pathian auhnak thawng ngai tuah, Ruahchannak in thla kan cam, Kan min tial cang kha kan hngalh, Thihnak tiva tan tikah, Jesuh thi hngatchan ca ah, "Na lam kha kan hmuhsak lai, Zul tuah," bianem in a ti.
