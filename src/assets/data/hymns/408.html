CEUNAK TLUNG KO SEH

Doh is G

1. Muihnak chung a um-mi sinah, Nangmah ceunak tlung ko seh, A kan Khamhtu Jesuh thawngin, Khamhnak kha an hmuh khawh lai.

CHO: Maw Bawipa, ceunak run pe ko, Rili ral khua zakip ah, Na dawtnak le vel-ngeihnak cu, man pek loin an hmuh lai.

2. Zan khuamui ngaih-chiatnak chungin, A tlaumi thlarau khamh tuah. A caan zong kha a dih zau lai, Rianrang in i tim ko tuah.

CHO: Maw Bawipa, ceunak run pe ko, Rili ral khua zakip ah, Na dawtnak le vel-ngeihnak cu, man pek loin an hmuh lai.

3. Nangmah nih na zultu hna cu, "Thawngthabia kha chim ko u, Nanmah sinah ka um zungzal," tiah bia kha na kan kam cang.

CHO: Maw Bawipa, ceunak run pe ko, Rili ral khua zakip ah, Na dawtnak le vel-ngeihnak cu, man pek loin an hmuh lai.

4. Maw kal lengmang Gospel teinak, Rili le khua zakip ah, Muihnak vialte ceu cang ko seh, A caan kha a luan hlan ah.

CHO: Maw Bawipa, ceunak run pe ko, Rili ral khua zakip ah, Na dawtnak le vel-ngeihnak cu, man pek loin an hmuh lai.
