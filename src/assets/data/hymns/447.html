CEUNAK RAM AH HIM IN

Doh is Ab

1. Mithiang mi hna kalnak lam kan zul ti, Kan Pathian nih lam a kan hruai, Vancung khua lei lam panh in kan kal i Deirel arfi kan cung a ceu.

CHO: Inn tha bik leiah kan kal lio, Lung-lawmhnak le nunnak co awkah; Kan phak tikah kan i lawm chin lai, Ngaihchiat um ti lai lo.

2. Lam tha a zul mithiang vialte bang khan, Zion khua ah kan kal cio lai, Bawipa kan hmuh tikah kan lawm chin lai, Zungzal amah he kan um lai.

CHO: Inn tha bik leiah kan kal lio, Lung-lawmhnak le nunnak co awkah; Kan phak tikah kan i lawm chin lai, Ngaihchiat um ti lai lo.

3. Lam tha zulh awk kha kan ruah cio ahcun, Pakhat le khat kan i daw lai, Pathian he kannih kan um ti tikah, Lunglawm in kan um zungzal lai.

CHO: Inn tha bik leiah kan kal lio, Lung-lawmhnak le nunnak co awkah; Kan phak tikah kan i lawm chin lai, Ngaihchiat um ti lai lo.

4. Thihnak tiva tih-nungmi pawngah khan, Kannih kha dir hmanh u sih law, Kan Pathian he cun kan tan ti lai i; A thang-thatnak hla kan sa lai.

CHO: Inn tha bik leiah kan kal lio, Lung-lawmhnak le nunnak co awkah; Kan phak tikah kan i lawm chin lai, Ngaihchiat um ti lai lo.
