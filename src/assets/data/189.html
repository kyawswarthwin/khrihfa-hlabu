KHRIH CAAH ZUMHTLAK SINAK

Doh is F

1. Ralchan nitlak nelrawn in hmelchunhnak a lang, "Khrih ca-ah zumhtlak si u," ti langhnak a si; Cu hla dawh cu aa hrawl, Fing le tlang kip hrawngin, "Khrih ca-ah zumhtlak si u," tinak hla dawh a si.

CHO: Nan tei hlan tiang, nan tei hlan tiang in, Kan Ralbawi ngan bik cu a au, A bia bang kan tuan lai. Ram cu kan lak cang lai, Bawi Khrih ca-ah zumhtlak si nak in kan tei cang lai.

2. Vawlei pumpi cul in an authawng cu ngai u, "Khrih ca-ah zumhtlak si u," ti langhnak a si; Tho law ral tha in tuan, Ralvengtu nih a ti, "Khrih ca ah zumhtlak si u," ti nak hla dawh a si.

3. Zumhtlakmi vialte he sualnak cu kan tei lai, "Khrih ca ah zumhtlak si u," ti langhnak a si; Satan ral hrat hmanhseh, Hla thiang kan sak hnawh lai,

4. Bawi Jesuh ke hram ah, Seino thazang chuah in, "Khrih caah zumhtlak sinak" in rian kan tuan lai; Thawngtha kan aupi lai, Vawlei pumpi cul in, "Bawi Khrih caah zumhtlak sinak" cu kan dirpi lai.
