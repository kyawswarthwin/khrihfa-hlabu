DAM TEIN

Doh is G

1. Kan chuahsemnak le a ram i dang hmanh seh, Bawi Khrih ah pumkhat te kan si, Bawipa dawtnak nih hmunkhat ah a kan hruai, lungkhat in tuan rel ti kan si.

CHO: Maw tonnak caan tha liamser hin hlah law, ti zongah tlaih khawh si tung lo, Kan tian ko kun lai, Dam tein, Mangtha u, Thlacam u, Tonthan um ko seh.

2. Nan kal ah siseh, nan tlun lei ah siseh, Bawipa nih in hup-pheng hna seh, Maw kan hawidawt hna, Then caan cu riangmang cang, nan zum in kan rung uai len lai.

CHO: Maw tonnak caan tha liamser hin hlah law, ti zongah tlaih khawh si tung lo, Kan tian ko kun lai, Dam tein, Mangtha u, Thlacam u, Tonthan um ko seh.

3. Vawlei ah thennak caan nih kan uk hmanh seh, Vanram ah thennak a um lo, Dam tein ti bia chim ti lonak ram cu kan ruahchan nak ram nuam a si.

CHO: Maw tonnak caan tha liamser hin hlah law, ti zongah tlaih khawh si tung lo, Kan tian ko kun lai, Dam tein, Mangtha u, Thlacam u, Tonthan um ko seh.
