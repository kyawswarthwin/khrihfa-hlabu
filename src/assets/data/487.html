LUNGKHAT TEIN TUAN U SI

Doh is G

1. Kan pupa zumhnak a hmun thlah loin i tlaih u si, an nun ning, cu chim kan duh, Lungkhat tein an rak um, Fel tein nun an i zuam, Bawipa bia bangin an i daw.

CHO: Sihmanhseh (Sihmanhseh tuchan) tu chan kan nunning zoh u, (kan nunning zoh u) Kan pupa nunning a leng ti lo, Ruah than caan a cu remh than caan a phan lungkhat te-in tuan u si.

2. Kan pupa lungput a tha kan lungput si cio ko seh, an nun ning cu chim kan duh, Hak-kauh depdet an duh lo, Zudin sa ei an rem lo, Nizarh ni tha tein an ulh.

CHO: Sihmanhseh (Sihmanhseh tuchan) tu chan kan nunning zoh u, (kan nunning zoh u) Kan pupa nunning a leng ti lo, Ruah than caan a cu remh than caan a phan lungkhat te-in tuan u si.

3. Kan pupa lungthin a thiang, Lungthiangmi si cio u si, an nun ning cu chim kan duh, Porhlawt ruamkai an duh lo, Mi nautat zong an hmang lo, Cawnpiaknak tha tein an zulh.

CHO: Sihmanhseh (Sihmanhseh tuchan) tu chan kan nunning zoh u, (kan nunning zoh u) Kan pupa nunning a leng ti lo, Ruah than caan a cu remh than caan a phan lungkhat te-in tuan u si.

4. Hringtu pa upat u si, Pathian mithmai cuang an si. An nun ning cu chim kan duh, Lungthin thazang dihlak in. Kan caah temhnak an in, Nunnak, fimnak an kan hmuhter.

CHO: Sihmanhseh (Sihmanhseh tuchan) tu chan kan nunning zoh u, (kan nunning zoh u) Kan pupa nunning a leng ti lo, Ruah than caan a cu remh than caan a phan lungkhat te-in tuan u si.
