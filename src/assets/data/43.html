A LIAN BIK PATHIAN TUM TUAH

Doh is G

1. Maw Siangpahrang lian bik, Lian bik Pathian tum tuah, Nangmah min kha, Kan in thangthat awkah, Kan daw tuah, Maw Pathian, Teinak le ngannak he, Kan huppheng tuah.

2. Maw Pathian Bia tum tuah, Teinak na vainam he, Na fale hna, Kan thlacamnak ngai law, Ral teinak, kan pe tuah, A thiangmi Jesuh Khrih, Kan sin tum tuah.

3. Maw Thiang Thlarau tum law, Kan lungthin ceuter tuah. Lam ding hmuhsak, Kan lung hnangam ko seh, Kannih he, va um tuah, Maw Thiang Thlarau kan chung, Rung lut tuah law.

4. Pa, Fapa, Thiang Thlarau, Thum komh Pathian pakhat, Nangmah sinah thangthat le sunparnak, Lianhngan le upatnak, Zungzal dong hin loin um camcin seh.
