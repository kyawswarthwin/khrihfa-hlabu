KUM AA MER LENGMANGMI AH HIN PATHIAN

Doh is G

1. Kum aa mer lengmang mi ah hin Pathian, Kan in bochan, Kan i ruahchan mi le tihmi chungah, Na kut kan hmuh, Ni thar le zing thar hna an chuah sinin, Na zaang-fahnak, A thar in kan hmuh t han.

2. Maw kan Pathian nangmah na kut chungah, Kan caan a um, Zumhnak in Biakam mi Ram ah kan hruai, Lam kan hmuhsak, Nangmah he cun, muihnak hi a ceu ko, Zumhnak muithlam, Hmuhnak ah canter ko.

3. Hmailei kum caan zong a uktu Pathian, Nang kan in zulh,Kan thawn lioah Bawipa kan kal tak hlah, Kan umpi ko, Nifa kan nunnak changreu rung si law, Kan nunnak dih tik i kan inn si ko.
